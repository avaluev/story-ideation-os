Go beyond the obvious 'twist ending' — include: structural revelations that reframe the entire narrative, character revelations that explain all prior behavior, setting revelations that change the stakes, temporal revelations that reorder events, moral revelations that reframe all prior judgments. Sources: cognitive film theory, Bordwell narrative analysis, Schmidhuber interestingness theory, Del Vecchio script analysis database.
